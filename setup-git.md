# Git Setup Commands for Bustan Al Reef Project

## After Git is installed, run these commands in PowerShell:

```bash
# Navigate to project directory
cd "C:\Users\97152\Desktop\al reef"

# Initialize Git repository
git init

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit: Bustan Al Reef cleaning website"

# Add remote repository (replace with your repo URL)
git remote add origin https://github.com/yourusername/bustan-alreef.git

# Push to GitHub
git push -u origin main
```

## If you don't have a GitHub repository yet:

1. Go to https://github.com
2. Create a new repository named "bustan-alreef"
3. Copy the repository URL
4. Replace the remote URL in the commands above

## Alternative: Use GitHub Desktop
1. Download GitHub Desktop from https://desktop.github.com
2. Install and sign in to your GitHub account
3. Click "Add an Existing Repository"
4. Select your "al reef" folder
5. Commit and push using the GUI
